CARPETA DE IMÁGENES
===================

Esta carpeta contiene las imágenes utilizadas en el sistema de Control de Accesos.

IMÁGENES REQUERIDAS:
--------------------

1. IngresoProveedores.png - Imagen para la tarjeta de "Proveedores" en el menú principal
   - Tamaño recomendado: 80x80 píxeles
   - Formato: PNG con fondo transparente

INSTRUCCIONES:
--------------

Coloque aquí su imagen "IngresoProveedores.png" para que aparezca en la tarjeta de Proveedores del menú principal.

Si la imagen no se encuentra, aparecerá un ícono de imagen rota en el navegador.
